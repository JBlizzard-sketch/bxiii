# BXII - Premium Smoke Culture Store

Vercel-ready static e-commerce site for BXII. Single-page, fully responsive, real-time cart functionality.

## Quick Start

```bash
vercel deploy
```

## Features

- Premium custom design with dark theme
- Responsive across all devices
- Real-time shopping cart
- Product filtering by category
- Mobile navigation
- WhatsApp integration for orders
- Smooth scroll animations

## Deployment

Push to GitHub, connect to Vercel. Automatic deploys on every push.

```bash
git push origin main
```

## Local Development

```bash
python3 -m http.server 3000
# Visit http://localhost:3000
```
